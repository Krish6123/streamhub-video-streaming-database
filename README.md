# StreamHub - Video Streaming Platform Database

## Overview

StreamHub is a DBMS academic project inspired by modern video streaming platforms such as Netflix.

The project focuses on designing a normalized relational database to manage users, subscriptions, movies, TV shows, watch history, ratings, watchlists, and content providers.

## Features

- User & Profile Management
- Subscription Plans
- Movies and TV Shows
- Seasons and Episodes
- Watch History
- Watchlists
- Ratings and Reviews
- Content Providers
- Universal Search
- Personalized Recommendations
- Provider Analytics

## Project Components

- Entity Relationship (ER) Diagram
- Relational Schema
- Database Design Documentation
- Advanced SQL Query Design

## Database Concepts Used

- Normalization
- Primary & Foreign Keys
- One-to-One Relationships
- One-to-Many Relationships
- Many-to-Many Relationships
- Joins
- CTEs
- Subqueries
- Aggregate Functions
